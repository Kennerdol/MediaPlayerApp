Good question 🙂

On GitHub (and in most open-source projects), the license is usually saved in a plain text file called LICENSE or LICENSE.txt.

Both work fine, but:

  • LICENSE (no extension) is the most common and recommended — GitHub automatically detects it and shows the license in the repo header.

  • LICENSE.txt is also valid, but slightly less common.

👉 Best practice: name it just LICENSE (all caps, no extension).
